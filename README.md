# Conception-Logicielle
Projet de conception logicielle orientée objet (ENSTA Brest) — Implémentation en Python et Java des concepts d'héritage, encapsulation et polymorphisme avec une démarche TDD (Test-Driven Development). Inclut 3 TPs : gestion d'étudiants, sécurisation des données et extension polymorphique avec des enseignants.
